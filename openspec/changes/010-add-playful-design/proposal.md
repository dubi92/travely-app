# Change: Add Playful Design Tokens & Components

## Why
The itinerary feature requires a more playful, whimsical visual design. This change extends the design system with soft gradients, glassmorphism effects, decorative elements, and playful animations to make trip planning feel fun and delightful.

## What Changes
- Add playful color palette (mint, peach, cyan gradients)
- Add glassmorphism card component with backdrop blur
- Add decorative background elements (clouds, stars)
- Add floating mascot widget with animation
- Add playful status badge component with emoji icons
- Add day tabs component (pill-shaped, scrollable)

## Impact
- Affected specs: MODIFY `design-system`
- Affected code: `lib/core/theme/`, `lib/shared/widgets/`
- Dependencies: `003-add-core-components`
- Blocks: `012-add-itinerary-screen`
